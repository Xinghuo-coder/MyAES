# Web界面使用指南

## 🚀 启动方法

### 方法 1：命令行启动（推荐）
```bash
python3 web_gui.py
```

### 方法 2：双击启动（macOS）
双击 `启动GUI.command` 文件

启动后会看到：
```
🔐 数据加密与密码管理系统 - Web界面
==================================================
🌐 访问地址: http://127.0.0.1:5000
⚠️  仅本地访问，不对外开放
🛑 按 Ctrl+C 停止服务
==================================================
```

在浏览器中打开：**http://127.0.0.1:5000**

## 📱 功能说明

### 1️⃣ 首次设置
- 首次访问时，系统会要求设置主密码
- 主密码用于保护所有加密数据，请务必记住
- 建议使用至少 12 位的强密码（包含大小写字母、数字和符号）

### 2️⃣ 登录
- 输入主密码登录系统
- 登录状态会在会话中保持
- 关闭浏览器后需要重新登录

### 3️⃣ 密码管理

**左侧 - 密码列表：**
- 显示所有已保存的密码
- 点击项目可选中
- 选中后可进行查看、复制、删除操作

**右侧 - 添加新密码：**
1. 填写网站/服务名称（必填）
2. 填写用户名（可选）
3. 填写密码（必填）
4. 点击"生成强密码"可自动生成安全密码
   - 可调整密码长度（8-32位）
5. 添加备注信息（可选）
6. 点击"保存密码"完成

**操作按钮：**
- **查看密码**：弹窗显示完整信息
- **复制密码**：一键复制到剪贴板
- **删除密码**：删除选中的密码（需确认）
- **刷新列表**：重新加载密码列表

### 4️⃣ 文件加密

**加密文件：**
1. 点击"选择文件"按钮
2. 选择要加密的文件（任何类型）
3. 点击"🔒 开始加密"
4. 加密完成后自动下载 `.encrypted` 文件

**解密文件：**
1. 点击"选择文件"按钮
2. 选择 `.encrypted` 文件
3. 点击"🔓 开始解密"
4. 解密完成后自动下载原始文件

**特性：**
- 支持拖放上传文件
- 自动显示选择的文件名
- 支持最大 100MB 文件
- 自动清理临时文件

### 5️⃣ 文本加密

**加密文本：**
1. 在"加密文本"区域输入要加密的内容
2. 点击"🔒 加密文本"
3. 加密结果显示在下方文本框
4. 点击"复制结果"可复制到剪贴板
5. 点击"清空"清除内容

**解密文本：**
1. 在"解密文本"区域粘贴加密文本
2. 点击"🔓 解密文本"
3. 解密结果显示在下方文本框
4. 点击"复制结果"可复制到剪贴板

### 6️⃣ 设置

**更改主密码：**
1. 输入当前密码
2. 输入新密码
3. 确认新密码
4. 点击"确认更改"

**关于信息：**
- 查看系统版本和技术信息
- 了解加密算法和安全特性

## 💡 使用技巧

### 浏览器兼容性
- ✅ Chrome/Edge（推荐）
- ✅ Safari
- ✅ Firefox
- ✅ 支持所有现代浏览器

### 密码管理
- 可以为不同网站保存多个密码
- 自动生成的密码安全性高
- 复制密码后建议使用完立即粘贴

### 文件加密
- 加密后的文件扩展名为 `.encrypted`
- 保留原始文件不会被删除
- 加密/解密过程在后台进行，大文件需要等待

### 文本加密
- 适合加密较短的文本内容
- 加密结果为 Base64 编码字符串
- 可以通过邮件、聊天工具安全传输

## 🔒 安全提示

### ⚠️ 重要提醒

1. **主密码安全**
   - 主密码遗失将无法恢复任何数据
   - 不要与他人分享主密码
   - 定期更改主密码

2. **本地运行**
   - Web界面仅在本地运行（127.0.0.1）
   - 不会对外网开放
   - 所有数据存储在本地 `vault/` 目录

3. **会话管理**
   - 使用完毕后记得点击"退出登录"
   - 关闭浏览器会自动清除会话
   - 不要在公共电脑上保存密码

4. **数据备份**
   - 定期备份 `vault/` 目录
   - 可以将备份存储在安全的云盘或U盘
   - 备份时确保包含所有 `.enc` 文件

## ❓ 常见问题

**Q: 无法访问 http://127.0.0.1:5000？**
A: 检查是否成功启动程序，确保端口5000没有被占用。

**Q: 上传文件失败？**
A: 检查文件大小是否超过100MB限制。

**Q: 忘记主密码怎么办？**
A: 无法恢复，这是为了确保安全性。请务必记住主密码。

**Q: 可以在手机浏览器上使用吗？**
A: 可以，在同一局域网内访问电脑的IP地址即可（需修改配置）。

**Q: 数据会上传到云端吗？**
A: 绝对不会，所有数据都存储在本地。

## 🛠️ 故障排除

### 程序无法启动
```bash
# 检查依赖是否安装
pip3 install -r requirements.txt

# 手动启动
python3 web_gui.py
```

### 端口被占用
如果5000端口被占用，可以修改 `web_gui.py` 最后一行：
```python
app.run(debug=False, host='127.0.0.1', port=5001)  # 改为5001
```

### 停止服务
在终端中按 `Ctrl+C` 即可停止服务
