# 数据加密与密码管理系统

一个安全的本地数据加密和密码管理工具，使用军事级加密标准保护您的敏感信息。

## 功能特性

- 🔐 **强加密算法**：使用 AES-256-GCM 加密标准
- 🔑 **密码管理**：安全存储和管理多个账户密码
- 📁 **文件加密**：加密/解密任意文件和文本数据
- 🛡️ **密钥派生**：使用 Argon2 密钥派生函数
- 💾 **本地存储**：所有数据加密后存储在本地，绝不上传云端

## 安全说明

- 主密码是访问所有数据的唯一凭证，请妥善保管
- 忘记主密码将无法恢复数据
- 加密数据存储在 `vault/` 目录
- 建议定期备份 `vault/` 目录

## 安装

```bash
pip install -r requirements.txt
```

## 使用方法

### 启动程序
```bash
python main.py
```

### 首次使用
1. 设置主密码（需要输入两次确认）
2. 主密码将用于保护所有加密数据

### 密码管理功能
- 添加新密码
- 查看已保存的密码
- 删除密码
- 复制密码到剪贴板

### 文件/数据加密
- 加密文本数据
- 解密已加密的数据
- 加密文件
- 解密文件

## 技术细节

- **加密算法**：AES-256-GCM（认证加密）
- **密钥派生**：Argon2（内存难度型，抗ASIC攻击）
- **盐值**：每次加密使用随机盐值
- **IV/Nonce**：每次加密生成新的初始化向量

## 注意事项

⚠️ **重要提示**：
- 这是一个本地加密工具，数据安全性取决于主密码强度
- 建议使用至少12位包含大小写字母、数字和符号的强密码
- 定期备份 `vault/` 目录到安全位置
- 不要将主密码告诉任何人
