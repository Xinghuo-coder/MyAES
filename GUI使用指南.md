# GUI 使用指南

## 启动方法

### 方法 1：双击启动（macOS）
直接双击 `启动GUI.command` 文件

### 方法 2：命令行启动
```bash
python gui_app.py
```

## 功能说明

### 1️⃣ 首次设置
- 首次启动时，系统会要求设置主密码
- 主密码用于保护所有加密数据，请务必记住
- 建议使用至少 12 位的强密码

### 2️⃣ 密码管理

**添加新密码：**
1. 点击菜单栏 `功能` → `密码管理`
2. 在右侧填写网站名称、用户名、密码
3. 可选：添加备注信息
4. 点击"生成强密码"可自动生成安全密码
5. 点击"保存密码"完成添加

**查看密码：**
1. 在左侧列表中选择一个密码项
2. 点击"查看密码"查看详细信息
3. 点击"复制密码"将密码复制到剪贴板

**删除密码：**
1. 选择要删除的密码项
2. 点击"删除密码"并确认

### 3️⃣ 文件加密

**加密文件：**
1. 点击菜单栏 `功能` → `文件加密`
2. 在"加密文件"区域：
   - 点击"浏览"选择要加密的文件
   - 指定加密后的保存位置（会自动建议文件名）
   - 点击"🔒 开始加密"
3. 等待加密完成提示

**解密文件：**
1. 在"解密文件"区域：
   - 点击"浏览"选择加密的文件
   - 指定解密后的保存位置
   - 点击"🔓 开始解密"
2. 等待解密完成提示

### 4️⃣ 文本加密

**加密文本：**
1. 点击菜单栏 `功能` → `文本加密`
2. 在"加密文本"区域输入要加密的文本
3. 点击"🔒 加密文本"
4. 加密结果会显示在下方，可点击"复制结果"

**解密文本：**
1. 在"解密文本"区域粘贴加密文本
2. 点击"🔓 解密文本"
3. 解密结果会显示在下方

### 5️⃣ 更改主密码

1. 点击菜单栏 `设置` → `更改主密码`
2. 输入当前密码和新密码
3. 确认更改

## 使用技巧

### 💡 文件选择
- 支持所有类型的文件加密
- 建议加密后的文件使用 `.encrypted` 扩展名
- 系统会自动建议输出文件名，也可以手动修改

### 💡 密码生成
- 点击"生成强密码"可创建安全的随机密码
- 可调整密码长度（8-32位）
- 生成的密码包含大小写字母、数字和符号

### 💡 数据备份
- 所有加密数据存储在 `vault/` 目录
- 建议定期备份该目录到安全位置
- 主密码哈希也存储在此目录中

## 安全提示

⚠️ **重要提醒：**
- 主密码遗失将无法恢复任何数据
- 不要在不安全的环境中使用
- 关闭程序时，加密数据会保持加密状态
- 密码复制到剪贴板后，建议使用后立即清空剪贴板

## 常见问题

**Q: 忘记主密码怎么办？**
A: 无法恢复，这是为了确保安全性。请务必记住主密码。

**Q: 可以在多台电脑上使用吗？**
A: 可以，只需复制 `vault/` 目录到其他电脑即可，但需要记得主密码。

**Q: 加密算法安全吗？**
A: 使用 AES-256-GCM 军事级加密标准，密钥派生使用 Argon2 算法。

**Q: 数据会上传到云端吗？**
A: 不会，所有数据都存储在本地，绝不上传。
